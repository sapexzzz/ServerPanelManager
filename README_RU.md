# Minecraft Server Panel Manager

Кроссплатформенная интерактивная панель управления Minecraft-сервером (Fabric или vanilla) в терминале. Использует **tmux** для работы сервера в фоне. Написана на Python 3.

---

## Возможности

| № | Функция |
|---|---------|
| 1 | Запуск / Остановка / Перезапуск / Принудительное завершение |
| 2 | Подключение к живой консоли сервера |
| 3 | Статус сервера (RUNNING / STOPPED) |
| 4 | Просмотр логов сервера в реальном времени |
| 5 | Бэкап мира с временной меткой (`.tar.gz`) |
| 6 | Менеджер модов — включить / отключить `.jar` файлы |
| 7 | Редактирование `server.properties` в выбранном редакторе |
| 8 | Изменение выделенной памяти (RAM) |
| 9 | Переключение авто-рестарта при падении сервера |
| 10 | Монитор ресурсов: CPU / RAM / аптайм в реальном времени |
| 11 | Удаление логов сервера или монитора |
| 12 | Менеджер белого списка (добавление / удаление игроков) |

## Требования

### Системные пакеты

| Пакет | Назначение |
|-------|-----------|
| `tmux` | Управление сессией сервера |
| `java` (21+) | Запуск Minecraft-сервера |
| `tar` / `gzip` | Бэкап мира (обычно уже установлены) |
| `nano` или `$EDITOR` | Редактирование `server.properties` |

**Debian / Ubuntu:**
```bash
sudo apt install tmux openjdk-21-jre-headless
```

**Arch Linux:**
```bash
sudo pacman -S tmux jdk21-openjdk
```

**Fedora / RHEL:**
```bash
sudo dnf install tmux java-21-openjdk-headless
```

### Python-пакеты

```bash
pip install -r requirements.txt
```

> `psutil` не обязателен, но рекомендуется. Без него монитор ресурсов использует системные команды (`free`, `top`).

## Настройка

1. **Скопируйте** проект в любую директорию, например `~/scripts/Panel_manager/`.

2. **Отредактируйте конфигурацию** в начале файла `menu.py`:

   ```python
   BASE_DIR  = Path.home() / "mc"          # путь к папке сервера
   JAR_NAME  = "fabric-server-mc.*.jar"    # точное имя jar-файла
   JAVA_RAM  = "2G"                        # выделяемая память
   ```

3. **Установите Python-зависимости:**

   ```bash
   pip install -r requirements.txt
   ```

4. **Сделайте скрипт исполняемым** (опционально):

   ```bash
   chmod +x menu.py
   ```

---

## Запуск

```bash
python3 menu.py
# или, если установлен бит исполняемости:
./menu.py
```

Навигация по меню — цифровые клавиши и **Enter**.

### Отключение от консоли сервера

Находясь в консоли tmux, нажмите **Ctrl-B**, затем **D** — вы вернётесь в панель.

## Ожидаемая структура директорий

```
<BASE_DIR>/
├── <JAR_NAME>          # jar-файл сервера
├── server.properties
├── whitelist.json
├── world/              # данные мира
├── mods/               # активные моды
├── disabled_mods/      # отключённые моды
├── backups/            # создаётся автоматически, архивы мира
└── server.log          # лог вывода сервера
```

---

## Примечания

- Панель работает в текущем терминале. Сам Minecraft-сервер запускается в отдельной detached-сессии tmux.
- Изменение RAM и авто-рестарта вступает в силу при **следующем** запуске сервера.
- Формат `whitelist.json` совместим с vanilla Minecraft (пары UUID + имя).
